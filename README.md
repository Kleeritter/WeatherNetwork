# WeatherNetwork

A collection of Esp8266 based Weatherstations sending data to a server via http.
# Hardware
## Used Parts
- Nodemcu
- DHT22
- BMP280
- MicroUSB Cable
- 14500 LI-Battery 
- Jumperwires
- Solarpanel
- Waveshare Solar Power Manager
